## Golf

> seb

Golf ball flight simulation using `rust` and `bevy` to run in the browser using `wasm`.


### Features

- Runs in browser
- Change clubhead loft, speed, spin
- Visualize in 3D world

### Build

```bash
cd sim
./bin/docker.sh
```
